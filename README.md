适用于 Windows 7 的非官方 Python 安装程序。

官方表述 Windows 7 不支持 Python 3.8 及以上版本。您可以使用非官方的 Windows 安装程序,以恢复对这些较旧版本的 Windows 支持。包含该文件api-ms-win-core-path-l1-1-0.dll等文件。

注意:在Windows 7 SP1 上,必须安装更新 KB2533623 才能运行。确保您的计算机通过 Windows 更新实现最新状态。或者,您可以手动安装 KB306388,取代 KB2533623。

对于每个Python版本,此仓库包含以下内容。
64位可执行安装程序(例如 python-3.8.0-amd64-full.exe)
32 位可执行安装程序(例如 python-3.8.0-x86-full.exe)
